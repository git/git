git-shortlog(1)
===============
v0.99.4, Aug 2005

NAME
----
git-shortlog - Summarize 'git log' output.


SYNOPSIS
--------
'git log --pretty=short | git shortlog'

DESCRIPTION
-----------
Summarizes 'git log' output in a format suitable for inclusion
in release announcements.
