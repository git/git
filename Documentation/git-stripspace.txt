git-stripspace(1)
=================

NAME
----
git-stripspace - Filter out empty lines


SYNOPSIS
--------
'git-stripspace' < <stream>

DESCRIPTION
-----------
Remove multiple empty lines, and empty lines at beginning and end.

OPTIONS
-------
<stream>::
	Byte stream to act on.
